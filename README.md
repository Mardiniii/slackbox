# README

This README documents the necessary steps to get SlashBox up and running.

Things you may want to cover:

## Ruby version
  2.2.4

## Database creation and initialization
  * `rake db:create`
  * `rake db:schema:load`

## Install dependencies
  * `bundle install`
  * `npm install`

## Development
  * `foreman start -f Procfile.dev`
